**Tasks** breaks the plan into an ordered checklist grouped **Setup → Foundational → user stories →
Polish**, gated so no story task starts before Foundational completes. Tasks are ordered by the story
dependency graph, tagged with their `US-#`, marked `[P]` when parallelizable, and each carries a
stable `T###` id and a concrete file path.

## Command

```bash
/amby.tasks
```
