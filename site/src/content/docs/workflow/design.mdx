**Design** produces the UI for each story: layout/wireframes, a component inventory (props, full
state set, token names), interaction criteria tied to `US-#`, and accessibility notes. Design tokens
are authored in three tiers — primitive → semantic → component. A **sign-off gate** makes the design
binding for every downstream phase once approved.

## Command

```bash
/amby.design
```
