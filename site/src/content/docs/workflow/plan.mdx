**Plan** is the first artifact where technology appears — the **HOW**. It sets the stack, libraries,
and constraints; maps each `FR-###`/`US-#` to how the design satisfies it; and honors every
constitution principle by number. Supporting artifacts (`data-model.md`, `contracts/`, `research.md`)
are produced only when needed.

## Command

```bash
/amby.plan
```
