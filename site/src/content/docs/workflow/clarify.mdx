**Clarify** resolves every open `[NEEDS CLARIFICATION: …]` marker the spec left behind. It asks one
focused question per marker, then patches only the affected lines in place — token-frugal and
non-destructive. When no markers remain, the spec is marked `ready`.

## Command

```bash
/amby.clarify
```
